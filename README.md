nosql-framework
===============

A Framework based on Lithium/MongoDB for Rapid Development of servers for Mobile Applications 

Basics:

	Create:
	
		URL:				api/index/

		requestJsonString:	{
		
								...
								
							}
						
	Retrieve:
	
		URL:				api/index/<id>
		
	Update:
	
		URL: 				api/index/<id>
		
		requestJsonString:	{
		
								...
								
							}
							
	Create a connection:
	
		URL:				api/index/<id>/<connection>
		
		requestJsonString:	{
		
								...
								
							}
